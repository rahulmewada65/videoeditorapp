# 🎬 VideoEditorApp

Lightweight mobile video editor built with **React Native** and **Firebase**.

## 👨‍💻 Developed by
**Rahul Mewada**

## 🛠️ Tech Stack
- React Native 0.79
- React Native Reanimated
- React Native Video
- Firebase
- Native Modules

## 🚀 Features
- Trim video
- Add text overlay
- Record or import videos
- Export in HD quality
